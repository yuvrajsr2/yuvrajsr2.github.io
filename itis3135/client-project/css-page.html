üé® What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML elements should be displayed on screen, paper, or in other media.

Example:

p {
  color: #00ffb3;
  font-size: 18px;
  text-align: center;
}

This code styles all paragraphs (`<p>`) to have bright green text, centered alignment, and a readable font size.

üß± CSS Selectors

Selectors define which HTML elements a style applies to.

  • Element selector: p { color: white; }
  • Class selector: .btn { background: #00ffb3; }
  • ID selector: #header { padding: 20px; }
  • Descendant selector: nav a { color: #e5f0ff; }

üìê CSS Layout (Flexbox & Grid)

Modern CSS uses Flexbox and Grid to align elements easily.

Flexbox Example:

.container {
  display: flex;
  justify-content: center;
  gap: 20px;
}

Grid Example:

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 15px;
}

Responsive design ensures your website looks great on any device size. Use media queries to adjust styles for smaller screens:

    font-size: 16px;
  }

  .card {
    max-width: 95%;
  }
}
